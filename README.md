# How to install
>
> In terminal :
>
> ```bash
> git clone https://github.com/Kenjy62/OCR_argentbank.git
> ```
>
> ```bash
> cd backend
> ```
>
> ```bash
> npm install
> ```
> ```bash
> npm run dev:server
> ```
>
> Back to /
>
> ```bash
> npm install
> ```
>
> ```bash
> npm start
> ```
